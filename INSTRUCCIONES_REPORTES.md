# Nuevos Indicadores Operativos

Se ha implementado el módulo de **Reportes** siguiendo la matriz de la imagen proporcionada.

## Acceso
1. Inicie la aplicación (`npm run dev`)
2. Navegue a la sección **Reportes** en el menú lateral.

## Indicadores Incluidos
### Central de Mezclas
- Pacientes fuera de programación
- Eficiencia Operativa
- Errores de preparación
- Reempaque de tabletería

### Asistencial
- Validación de prescripciones
- Asesoría farmacológica
- Reporte de RAMs
- Gestión de Alertas

### Farmacia
- Atención por canal (Ventanilla, Domicilio, Sala)
- Cumplimiento de entregas
- Respuesta a requerimientos

**Nota:** Los datos actuales son simulados. Para conectar con datos reales, se requiere una fuente de datos (Excel o Base de Datos) que contenga estos registros diarios manuables.
